## 段落和换行

一个 Markdown 段落是由一个或多个连续的文本行组成，它的前后要有一个以上的空行。普通的 Markdown 段落 **不可以** 用空格或制表符来缩进。

> 空行：一行之内只包含回车换行、空格、制表符。

- **例如**：

    ```
    这是第一段话

    这是第二段话
    ```

	显示的效果如下：

    这是第一段话

    这是第二段话

- **如果两段之间不空一行，则无法正确分段。反面例子：**

    ```
    这是第一段话
    这是第二段话
    ```

	**显示效果如下：**

	这是第一段话
    这是第二段话

## 段落内部换行

在行末输入 **两个空格** 可以达到比分段更加紧凑的换行的效果。

- **例如：**

    ```
    这是第一行  
    这是第二行
    ```

    （请注意“这是第一行”后的两个空格，可以用鼠标拖动选中它们）

    **显示效果如下：**

    这是第一行  
    这是第二行

## 段落多换行

以下内容为 Wxygen 写，如有巧合纯属雷同。

由于只含有普通的空格、Tab 制表符的行都会被标记为空行，因此想要实现多个换行可以插入一些别的奇奇怪怪的东西。

**比如 $\LaTeX$ 的空格。**

示例：

- **【推荐】使用行内 \\ 进行换行，**
  
    ```
    这是第一行

    $ \ $

    这是第二行
    ```

    **显示效果如下：**

    这是第一行

    $ \ $

    这是第二行

- **使用 \\\\ 进行换行，**

    ```
    这是第一行

    $ \\ $

    这是第二行
    ```

    **显示效果如下：**

    这是第一行

    $ \\ $

    这是第二行

- **使用行间 \\ 进行换行，**

    ```
    这是第一行

    $$ \ $$

    这是第二行
    ```

    **显示效果如下：**

    这是第一行

    $$ \ $$

    这是第二行
